About Us

Welcome to our Weather Application! This project is designed to provide real-time weather updates for cities around the world. Using cutting-edge technologies and APIs, we aim to deliver accurate and reliable weather information to our users.

Our Mission

Our mission is to simplify access to weather data and empower users to make informed decisions based on real-time weather conditions.

Features

  • Real-time weather updates
  • City-based weather search
  • Responsive design for all devices
  • Detailed weather parameters including temperature, humidity, wind speed, and more

Technologies Used

  • Python Flask Framework
  • HTML5, CSS3, and JavaScript
  • Weather API Integration
  • Responsive Web Design
